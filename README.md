## 这个仓库是什么？

`yueda-ai-workflow-kit` 是悦达智科用于公开沉淀 AI 工作流、Agent Skill、内容自动化与人机协同方法的样板库。

它的目标不是炫技，而是帮助中小企业把日常工作中反复发生的任务、判断、沟通和复盘，整理成可复制、可执行、可持续迭代的 AI 工作流。

简单说：

> 把企业经验变成流程，把流程变成 Skill，把 Skill 变成可协同的 AI 生产力。

## 当前公开样板

### 1. 公众号资料包生成 Skill

用于把一个选题，整理成适合公众号发布的完整资料包，包括：

* 标题
* 摘要
* 正文结构
* 公众号排版建议
* 图片清单
* 插图位置
* 发布检查清单

目标不是替代创作者，而是让内容生产过程更稳定、更可复用。

### 2. 企业问题诊断 Skill

用于帮助企业管理者判断一个问题是否值得投入资源，识别真正的 A 类问题。

### 3. 短视频脚本策划 Skill

用于生成短视频选题、开头钩子、口播稿、分镜建议、封面文案和平台发布版本。

## 什么是 Agent Skill？

在本项目中，Agent Skill 指的是一套可以被 AI 使用的专业任务说明包。

它通常包括：

* 角色定位
* 任务目标
* 输入要求
* 输出格式
* 判断标准
* 执行步骤
* 示例输入
* 示例输出
* 风险提醒

它不是一句提示词，而是一套让 AI 稳定完成某类任务的工作说明书。

本仓库只公开基础样板、学习示例和通用框架。

* 客户数据

* 完整自动化脚本

* 商业版 Agent Skill 包

如果你需要完整企业版工作流、私有化部署或定制化 Skill 设计，需要通过商业合作方式进行。

## 商业服务方向

悦达智科可围绕以下方向提供服务：

### 1. 企业 AI 工作流诊断

帮助企业判断哪些任务最适合优先用 AI 提效。
