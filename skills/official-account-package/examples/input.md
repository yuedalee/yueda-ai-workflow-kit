# 示例输入｜公众号资料包生成 Skill

> 这个文件用于演示如何调用「公众号资料包生成 Skill」。

---

## 使用目标

请使用「公众号资料包生成 Skill」，基于下面这个选题，生成一套适合微信公众号发布的资料包。

---

## 输入信息

```text
选题：企业为什么不能只靠买 AI 工具实现降本增效？

目标读者：中小企业老板、企业高管、内容团队负责人

发布平台：微信公众号

文章目的：
教育市场，建立悦达智科在企业 AI 工作流和 Agent Skill 设计方面的专业认知。

核心观点：
AI 的价值不在工具本身，而在于把企业中反复发生的任务、判断、沟通和复盘，重构为可执行、可复用、可迭代的工作流。

已有素材：
悦达智科正在搭建企业 AI 工作流和 Agent Skill 公开样板库，目标是帮助中小企业把经验变成系统，把系统变成生产力。

希望风格：
专业、理性、面向老板和管理者，但不要太技术化；语言要清楚、有判断、有商业洞察。

是否需要配图建议：是

是否需要生成公众号摘要：是

是否需要生成 HTML 排版建议：是
```

---

## 期望输出

请输出以下内容：

```text
01_公众号标题候选
02_公众号摘要
03_正文纯文本稿
04_正文结构说明
05_配图清单与插入位置
06_金句卡建议
07_封面图建议
08_发布前检查清单
09_HTML 排版结构建议
10_机器可读 JSON
```

---

## 输出要求

* 不要编造客户案例。
* 不要虚构具体数据。
* 不要输出 API Key、Webhook、服务器信息等敏感内容。
* 不要把内部执行说明直接写进面向读者的正文。
* 正文要适合微信公众号阅读。
* 文章开头要有明确冲突。
* 结尾要有品牌观点收束。
